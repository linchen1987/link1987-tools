# Link1987 Tools

A curated collection of simple, elegant, and useful tools for developers to boost productivity.

## Features

- **Clean Interface**: Built with modern UI components for a distraction-free experience.
- **Developer Focused**: Tools designed specifically for common development tasks.
- **Fast & Responsive**: Powered by Next.js for optimal performance.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## Getting Started

First, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
